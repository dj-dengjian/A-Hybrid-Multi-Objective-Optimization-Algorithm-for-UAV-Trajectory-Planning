# A-Hybrid-Multi-Objective-Optimization-Algorithm-for-UAV-Trajectory-Planning
A Hybrid Multi-Objective Optimization Algorithm for UAV Trajectory Planning
* **1. Core MS-INSGA2 Algorithm:** The complete implementation of the proposed hybrid algorithm, featuring the linear interpolation initialization, dynamic guidance strategy, adaptive crossover/mutation, and Simulated Annealing (SA) based local search mechanism.
* **2. Ablation Studies (Section 5):** Ready-to-run scripts to replicate the performance comparisons between MS-INSGA2 and its single-strategy variants:
    * `G-INSGA2`: NSGA2 with only the Guidance strategy.
    * `A-INSGA2`: NSGA2 with only the Adaptive crossover/mutation strategy.
    * `I-INSGA2`: NSGA2 with only the Interpolation initialization.
    * `L-INSGA2`: NSGA2 with only the Local search strategy.
